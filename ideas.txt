محاور المحتوى والتخصص
 * مجال التخاطب: تقديم إرشادات وتمارين لدعم النطق واللغة
 * تنمية المهارات: أدوات وأنشطة لرفع القدرات الإدراكية والحركية
 * تعديل السلوك: استراتيجيات علمية للتعامل مع التحديات السلوكية للأطفال

 * نظام الكوبونات: بيع كوبونات خصم حصرية لخدمات المراكز (مثل مركز لايف سكيلز) ومنتجات الوسائل التعليمية (مثل إيديو ماركت)

 * التسويق التبادلي: العمل كحلقة وصل بين مقدمي الخدمة والمستفيدين لضمان تدفق العملاء للمراكز المشاركة

Comprehensive Development Prompt for Platform Maza (منصة ماذا)
Project Overview:
Develop "Platform Maza" (منصة ماذا), a complex, multi-functional digital ecosystem supporting Arabic-speaking families of people with special needs. This is a progressive web application combining telehealth, AI-driven diagnostics, educational content, a marketplace, and dynamic pricing.
Core Role-Based Access:
 * Admin: System management, content validation, data analytics.
 * Specialist/Center: Manage profiles, schedule sessions, conduct telehealth, access child records.
 * Parent (End User): Book sessions, take AI assessments, manage child's IEP (Individualized Education Program), purchase coupons.
Key Functional Specifications & Technical Modules:
1. Localization & Dynamic Payment System (Geo-IP Based):
 * Implement a payment gateway (e.g., Paymob, Stripe, PayPal) that automatically detects the user's location via IP Address.
 * Pricing Logic:
   * Inside Egypt: Displays prices in EGP and provides local payment options (Fawry, Vodafone Cash, local cards).
   * Outside Egypt: Displays prices in USD and provides international payment options (Credit Cards, PayPal).
2. AI-Powered Assessment & Reporting Module:
 * Integrate a flexible assessment interface to administer standard tests (Speech, Skill, Behavior, Psychometric, Intelligence (IQ), Language).
 * AI Integration Task: Develop/integrate an AI model (e.g., via OpenAI API) to:
   * Analyze test scores (manual input or interaction data).
   * Automatically generate a comprehensive Diagnostic Report (in Arabic).
   * Automatically create a customized Individualized Education Program (IEP/البرنامج العلاجي الفردي) with short-term and long-term goals based on the diagnosis.
3. Telehealth & Booking System:
 * Implement a real-time scheduling calendar (like Calendly) where specialists set availability.
 * Integrate a secure, HIPAA-compliant video conferencing API (e.g., Agora, Twilio Video, or Zoom API) for online sessions within the platform.
 * Automatically send WhatsApp/Email/SMS session reminders.
4. Progress Tracking & Case Management:
 * Create a "Digital Case File" for each child.
 * Allow specialists to log progress notes after each session.
 * Include a data visualization dashboard (charts/graphs) showing the child's progress against IEP goals over time.
5. Business Model Modules:
 * Coupon System: Generate unique QR/Codes for discounts on partner services (like Life Skills Center) or products (like EduMarket), with safe checkout.
 * Data Acquisition & Lead Generation: Implement a robust database to securely collect user data (parent profile, child’s condition) to connect them with suitable centers.
 * Center Directory & Rating System: A searchable map/list of centers with an approved-only rating/review mechanism.
6. Content Management & Specialties:
 * A blog/content hub categorized by: Speech, Skill Development, Behavior Modification, Psychology, etc.
Technical Stack Recommendations (to be confirmed by the developer):
 * Backend: Node.js/Python (Django or FastAPI) for robust AI integration and API management.
 * Frontend: React/Vue.js (Responsive for mobile).
 * Database: PostgreSQL for structured medical-grade data.
 * Security: Implement strict data encryption and privacy protocols (GDPR/HIPAA guidelines).
Expected Output:
Please provide a full technical architecture plan, database schema, API documentation structure, and the core boilerplate code for the Dynamic Pricing and AI-Reporting modules.

- *AI لتوضيح الكلام وتعزيز التدريب*: أنظمة تعلم آلي بتحول الكلام غير الواضح (مثل حالات الـdysarthria) إلى كلام مفهوم وتقلل أخطاء الكلمات بحوالي 28%، وكمان بتولد تمارين فورية وتقييم آلي لحالات التأخر والنطق. أدوات AI بقت تُستخدم لتتبع التقدم وتقليل عبء التوثيق على الأخصائيين. 【a458】【b01a】

- *أجهزة قابلة للارتداء*: جهاز مرن بيتلزق على الرقبة ويترجم حركات عضلات الحنجرة لكلام مسموع بدقة ∼95%، بيساعد اللي فقدوا الصوت بعد عمليات أو أمراض. 【cc14】

- *التقنية في الجلسات اليومية*: 89% من الأخصائيين بيستخدموا أدوات رقمية أسبوعيًا (Boom Cards، فيديوهات، ملفات تفاعلية، شاشات ذكية)، وده زوّد التفاعل وسهّل المتابعة. منصات Tele-therapy الآمنة (HIPAA) بقت معيار للجلسات عن بُعد. 【8a72】【9eb5】

- *الفصول الافتراضية والعلاج عن بُعد*: دراسات حديثة رصدت توجه كبير لاستخدام الفصول الافتراضية وأدوات التعلم المعزز بالتكنولوجيا في علاج اضطرابات التواصل. 【0d83】【172d】

- *بروتوكولات جديدة داخل فرق طبية*: التخاطب دخل ضمن فرق علاج اضطرابات النوم (مثل انقطاع النفس) كخيار بديل لما الطرق التقليدية مش كافية. 【37bb】
